About the project

This project was created in the Praxis der Forschung event on the way to MSc. at the KIT at TECO.

General Information

SceneManager is a Web Repository for uploading, storing and retrieving Google Tango Area Description Files. The service is open to the public. With SceneManager you can upload your recoded ADF and tag it with information for localization. Examples areGPS coordinates, the altitude where the ADF file is recorded and its level if its inside a building. Furthermore, you can add an name and a description for your ADF. To add overlaying annotation information into an ADF scene additional data is required. SceneManager allows to upload Scene Data associated with the ADF file. Scenes are JSON formatted files that can be edited manually or graphically using a Tango device and the EditSys app.

Overview

This tab gives you an overview of all the uploaded Scenes and ADFs by other users. To make it easy to find uploaded files around a specific place, you can search with the map on the right side. For detailed information you can see the table. The following fields are included in the data.

Type
Specifies whether the file is just an ADF or an Scene created by the edit system
Name
Name of the uploaded file. Could be room number or anything else meaningful
Description
In addition to name, you can add a more detailed description here
Latitude
Latitude of the GPS coordinate to localize the file
Longitude
Longitude of the GPS coordinate to localize the file
Altitude
Altitude at the place the file was recorded. Useful for e.g. high Buildings.
Level
Direct assignment of a building level

In order to download the Scene or ADF, you can click on the download button in the table and receive the file directly from SceneManager.

Manage

To reach this tab you need to be signed in with you Google account. Here you can see your own uploaded Scenes and ADF files in the list as well on the map on the right side. If you wish to remove a file, you can click the delete button to do so.

Upload

Here you can upload your ADF files and scenes. To be able to upload you also have to be signed in with your Google account. To upload a file you have to fill out all the fields in the form on the left side. To help you to find your location you can use the “Locate me” button to get the rough location from your browser. You also can use the map on the right side to choose your location or the make it more accurate. All fields are mandatory, except altitude, since this can be hard to figure out. By default this value is set to 0.0 if left out.

SceneManagerClient

To upload and download ADF-Files from Scene Manager we recommend you to use the SceneManagerClient App. It can be found here and allows you easy management of your ADF-Files across multiple devices and users.

Upload

1 Recoding of ADF-File

The first step is to record an ADF-File of the room you want to use Area Learning in. In our how-to video we use the App Tango Explorer, which can be found here. (http://projectango.wikidot.com/apps)

2 Export your ADF-File

In order to upload the recorded ADF-File from step 1, you need to export it on your file system. This can be done with the ADF Tool(https://play.google.com/store/apps/details?id=net.chucknology.tango.adftool). To export a file, simply click on the item in the list you want to export. Make sure you selected the right category in the top bar. Once exported you can switch to File System and your exported ADF-File should appear. The ADF-Files are exported to /localStorage/ADFFiles.

3 Upload file to Scene Manager

Once your file is exported you can use the Scene Manager Client to upload the file to Scene Manager. Just select Upload on the start screen. Then you can enter all the meta information about the ADF-File like name, description, location, altitude and level. For Location you can locate yourself by clicking the “Location” button. Select the file you wish to upload by clicking “Select File”. After all needed Information is entered, you can upload your file clicking the upload button. In order to authenticate you need to use your Google account.

Download

1 Select File to Download

Open the Scene Manager Client and choose Download. Then you find a mask to search the area you are interested in. By clicking “Location” you can locate yourself. Use the slider to choose the radius to search around you. After clicking “Find” you should see entries in the list at the bottom that match your search. To download the file, select it from the list. Now wait till your file is downloaded. You will find your file in /localStorage/ADFFiles.

2 Import downloaded ADF-File

To use your new downloaded ADF-File in an Tango Application you need to import it. This functionality is provided by the ADF-Tool (https://play.google.com/store/apps/details?id=net.chucknology.tango.adftool). In the upper bar select “Storage”. Now you see a list of local files on your device. To import your new downloaded file, select it from the list. To check if it worked, you can switch on the bar to ADFs and you should see your new imported file in the list.

EditSys

EditSys is a proof of concept Android app for overlaying augmented objects (POIs in form of points, planes and cubes) into the real world using the technologies provided by Google Tango. Such created areas with additional objects we call a “Scene”.

General Information

In order to use EditSys you need to record an ADF File of the room you want to place objects with EditSys. For this, we recommend the Contstructor App provided by Google. After recoding the area and savingthe ADF you can open EditSys and load a new Scene. After EditSys has localized itself you are ready to use the functions of EditSys explained in the next chapter.

Functions

The following table shows the function provided by the EditSys

Place POI

After localization you can place objects by clicking on a point in the real world. Then a mask for entering details about the POI is shown to select description, color etc.

Edit POI

By long clicking on a POI you get a context menu where you are able to delete or edit the POI, as well as trigger an web-call if this was associated.

If you select a POI you get a control panel to move the selected POI as well as changing its orientation. By clicking the Edit option you can change description, color and so on

Upload Scene

By clicking upload Scene you can upload the current state of your EditSystem with the placed Objects to the SceneManager

Download Scene

With this function, you can download directly a previously recorded scene from SceneManager to your device. You can search your area for existing Scenes, which you can choose from.
